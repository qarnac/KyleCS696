General Description

Designed for students beginning to learn grammar and elementary arithmetic including addition, subtraction, multiplication, and division as well as an additional category of prime numbers.

The grammar portion of the game includes three categories which include nouns, verbs, or adjectives.

The math portion gives players the ability to choose from three difficulty levels which are easy, intermediate, and hard. Game variables can also be altered which affect how the game is played such as player speed, the number of possible answers, and the presence of an enemy player.

How to Play

Game Details

For math based problems, the number range will vary for each problem type depending upon the difficulty level chosen. Subtraction will always result in a non-negative value, and the quotient in a division problem will always be a whole number. The following list provides the problem type followed by the number range for easy, intermediate, and hard difficulty:

  • Addition (0-6 + 0-6, 0-20 + 7-20, 0-30 + 21-30)
  • Subtraction (0-8 - 0-8, 15-20 - 0-14, 21-35 - 0-20)
  • Multiplication (0-5 x 0-5, 4-9 x 6-9, 5-12 x 9-12)

Play Math Game

For grammar based problems, a sentence will be provided with the ability to click on any of the available words. The player will be asked to click on a number of either nouns, verbs, or adjectives in the sentence before advancing to the next question. Here are some examples of nouns, verbs, and adjectives.

  • Nouns (car, baseball, afternoon)
  • Verbs (run, behave, concentrate)
  • Adjectives (red, boring, embarrased)
